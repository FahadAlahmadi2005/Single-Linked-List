# Singly Linked List Implementation in Java

This repository contains a clean and efficient implementation of a **Singly Linked List** from scratch. This project is part of my journey in mastering **Data Structures & Algorithms (DSA)** with an emphasis on **Software Engineering** best practices.

---

## 🚀 Features

* **Generics Implementation:** Supports any data type (Integers, Strings, Custom Objects).
* **OOP Focused:** High encapsulation with a private inner Node class.
* **Comprehensive Operations:**
    * `insertFirst(E data)`: Insert at the beginning.
    * `insert(E data)`: Append to the end.
    * `remove(E key)`: Remove a node by its value.
    * `display()`: Print the entire list.
    * `isEmpty()` & `size()`: Utility methods for state checking.

---

## 📊 Complexity Analysis

| Operation | Time Complexity | Space Complexity |
| :--- | :--- | :--- |
| Insertion (at Head) | $O(1)$ | $O(1)$ |
| Insertion (at Tail) | $O(n)$ | $O(1)$ |
| Search / Access | $O(n)$ | $O(1)$ |
| Deletion | $O(n)$ | $O(1)$ |
